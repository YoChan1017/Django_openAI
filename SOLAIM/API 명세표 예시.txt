#챗봇 요청
sol/chatbot/			POST		여행지와 여행일 수를 받은 챗봇 생성
{
    "location": "여행지",
    "days": 여행기간
}
{
   "session_id": " abc ",
   "response": " 안녕하세요! (선택한 여행지)에서 (선택한 여행기간)일 동안의 여행을 도와드릴게요. 궁금한 점이 있으면 말씀해주세요!"
}
